# 指尖上的中国 — 需求文档

## 一、项目概述

**项目名称：** 指尖上的中国 · 非遗体验课在线预约平台

**项目定位：** 连接非遗传承人与爱好者的 O2O 体验课预约平台。用户可在线浏览非遗项目、预约体验课程、上传作品展示、参与非遗投票；管理员可审核作品、管理内容、发送通知。

**目标用户：**
- 普通用户：对非遗文化感兴趣，想预约体验课，展示学习成果
- 管理员：非遗传承人或平台运营方，审核内容、管理用户、发布通知

## 二、功能需求

### 2.1 用户认证模块

| 功能 | 描述 |
|------|------|
| 注册 | 用户输入用户名、密码（6位以上）、选择头像 emoji |
| 登录 | 用户名+密码验证，登录后保持状态 |
| 退出 | 清除登录状态，返回登录页 |
| 持久化 | 刷新页面不丢失登录状态（localStorage） |

### 2.2 非遗浏览模块

| 功能 | 描述 |
|------|------|
| 首页展示 | 9 个非遗分类以卡片形式展示（剪纸、皮影戏、刺绣等） |
| 分类详情 | 每个分类有介绍、文章列表、视频列表 |
| 文章阅读 | 点击文章弹出详情弹窗阅读全文 |
| 视频观看 | 点击视频弹出播放弹窗（链接到 B 站） |
| 动态内容 | 管理员可在后台新增文章和视频，前端自动展示 |

### 2.3 预约体验模块

| 功能 | 描述 |
|------|------|
| 选择项目 | 下拉选择非遗体验项目 |
| 选择时间 | 日期选择 + 时间段选择（已占用的自动禁用） |
| 填写信息 | 人数、姓名、电话 |
| 提交预约 | 校验完整性后提交，成功后弹出确认弹窗 |
| 查看预约 | 个人中心查看自己的预约记录 |

### 2.4 作品展示模块

| 功能 | 描述 |
|------|------|
| 上传作品 | 选择非遗项目 + 拖拽/点击上传图片 + 写描述 |
| 图片处理 | 自动压缩到 500px，JPEG 格式 |
| 作品展示 | 网格布局展示所有作品 |
| 删除作品 | 用户可删除自己的作品 |
| 作品审核 | 管理员审核作品（通过/拒绝） |

### 2.5 非遗地图投票模块

| 功能 | 描述 |
|------|------|
| 中国地图 | 真实地理数据绘制中国轮廓，标注 31 个省份非遗红点 |
| 悬停查看 | 鼠标悬停红点显示该省非遗项目和投票数 |
| 投票 | 点击"我感兴趣"按钮投票，每个用户每项限投一票 |
| 排行榜 | 首页底部显示热门非遗投票排行 |

### 2.6 管理后台模块

| 功能 | 描述 |
|------|------|
| 数据概览 | 用户数、预约数、作品数、待审核数、内容数统计 |
| 用户管理 | 查看所有用户，展开看每个人的预约和作品 |
| 预约管理 | 查看全部预约记录 |
| 作品审核 | 待审核/已通过/已拒绝三个 tab，可批准或拒绝 |
| 内容管理 | 新增/编辑/删除文章和视频，关联到非遗分类 |
| 消息通知 | 发送通知给单个用户或全站广播 |

### 2.7 消息通知模块

| 功能 | 描述 |
|------|------|
| 发送通知 | 管理员选择目标用户（或全站），填写标题和内容发送 |
| 接收通知 | 用户在"我的→消息" tab 查看通知列表 |
| 已读标记 | 用户可标记已读，未读通知有红点和左侧红色边框 |

## 三、非功能需求

| 需求 | 说明 |
|------|------|
| 响应式适配 | 支持桌面端和移动端，900px 断点自适应 |
| 页面过渡 | 页面切换有淡入淡出动画 |
| 数据安全 | 密码明文存储（本阶段为课程项目，生产环境需加密） |
| 权限控制 | 普通用户和管理员角色分离，管理端操作需 is_admin 权限 |
| 部署访问 | 通过 GitHub Pages 公网访问 |
